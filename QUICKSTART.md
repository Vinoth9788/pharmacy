# HealthCare Pharmacy Website

A modern, responsive pharmacy website with AI-powered chatbot support.

## Quick Start

1. Open `index.html` in your web browser
2. Or use a local server:
   ```bash
   python -m http.server 8000
   ```
3. Navigate to `http://localhost:8000`

## Customization

### Update Medicines
Edit `data/medicines.json`

### Change Colors
Modify CSS variables in `css/style.css`

### Chatbot Responses
Edit `js/chatbot.js`

## Deployment

3. Done!
